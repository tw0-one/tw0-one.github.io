方法一

首先安装whell

进入下面这个网址:

https://www.lfd.uci.edu/~gohlke/pythonlibs/

下载与自己电脑版本对应的.whl文件(不要该文件名!)

输入python –version查看python版本,输入systeminfo查看电脑位数:

本机python版本为3.9，64位的。

下载与之对应的版本(保存在哪里都一样):

进入保存.whl所在的文件夹，执行pip install lxml-4.6.3-cp39-cp39-win_amd64.whl 此处的文件为你所下载的文件名。

安装成功后，用pip install python-pptx 安装就行了。

方法二:

在Pycharm上安装pptx，打开设置

然后进入Projiect里面的interpreter点击加号

会出现下面界面，安装图中的包就行了(右下角有install package)

其实这种方式最终也会用到pip，只是这种是图形化的

参考连接:https://www.jianshu.com/p/ecc187c2a46a
